⌨️ 필수 단축키 (PC)

Enter          새 줄 생성
Tab            들여쓰기 (자식으로)
Shift+Tab      내어쓰기 (부모로)
@              날짜 입력
Backspace      빈 줄에서 누르면 삭제

📱 모바일: 들여쓰기는 점(●)을 한 번 탭하세요

🎯 핵심 기능

  • 📅 날짜 입력: @ 입력 → 달력 팝업 → 날짜 선택 (예: 📅 01-15 (수))
  • 🔒 행 보호: 호버 → 🔓 클릭 → 보호 활성화 (편집/삭제 불가)
  • ✏️ 텍스트 서식: 텍스트 선택 → 자동 팝업 (볼드/이탤릭/밑줄/취소선/형광펜 8종/글자색 10종)
  • 🔍 검색: 상단 검색창 입력 → 실시간 결과 → ↑↓ 또는 Enter로 이동
  • 🎯 드래그 이동:
    → PC: 체크박스와 별 사이의 검은 점(●)을 마우스로 클릭하고 드래그
    → 점 위에 마우스를 올리면 파란색으로 커집니다
    → 모바일: 점을 0.5초 길게 누른 후 이동
    → 모바일 들여쓰기: 점을 한 번 탭하면 하위 레벨로 이동

☁️ 클라우드 동기화 (중요!)

📌 첫 설정: [Drive 연결] 클릭 → Google 로그인 → 권한 승인
  • 30초마다 자동 저장
  • 10초마다 최신 버전 확인
  • 다른 기기: 같은 Google 계정 로그인 → [불러오기] 클릭
  • 오프라인 사용 가능 (로컬 저장)

⚠️ 반드시 알아야 할 사항

1. 보호된 항목 (🔒): 편집/삭제 불가 → 먼저 🔓로 해제 필요
2. 날짜 형식: 연도 표시 안 됨 (월-일만, 예: 01-15 (수))
3. 드래그 제한: 부모를 자식으로 이동 불가 (순환 방지)
4. 자동 저장: 30초마다 자동 저장 (손실 걱정 NO)

💡 유용한 팁

  • Bullet (•) 클릭 → 해당 항목만 집중 보기
  • 중요 항목은 🔒 보호로 실수 방지
  • 완료된 작업도 🔒 걸어서 보존
  • @날짜로 마감일 표시
  • 색상으로 우선순위 구분

🚨 문제 해결

편집 안 됨    → 🔒 확인 후 해제
동기화 안 됨  → [불러오기] 클릭
내용 사라짐   → [불러오기]로 복구
