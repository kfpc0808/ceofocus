# 📅 일정관리 시스템 - Phase 1

KFPC 보험 컨설턴트를 위한 지능형 일정관리 프로그램

---

## ✨ 주요 기능 (Phase 1)

### ✅ 완성된 기능

1. **조르테 스타일 캘린더**
   - 주간 뷰 (timeGridWeek) - 메인
   - 월간 뷰 (dayGridMonth)
   - 일간 뷰 (timeGridDay)
   - 리스트 뷰 (listWeek)

2. **일정 관리 (CRUD)**
   - 빠른 추가 (빈 칸 클릭)
   - 상세 추가 (모달 폼)
   - 드래그로 날짜 변경
   - 일정 수정/삭제
   - 중요 표시 (⭐)
   - 완료 체크 (✅)

3. **구글 드라이브 자동 저장**
   - AES 암호화
   - 3초 디바운스 자동 저장
   - 실시간 동기화
   - 데이터 보안

4. **검색 & 필터**
   - 제목/고객명 검색
   - 타입별 필터
   - 상태별 필터 (완료/미완료)
   - 중요 일정만 보기

5. **인쇄 기능**
   - 캘린더 인쇄
   - 프린트 최적화

6. **설정**
   - 타입별 색상 설정
   - 기본 뷰 설정
   - 시간 범위 설정

7. **모바일 반응형**
   - 터치 친화적
   - 하단 네비게이션
   - 큰 폰트 (가독성 최우선)

## 🚀 빠른 시작

### 1. 파일 업로드

GitHub 또는 Netlify에 업로드:

```bash
# GitHub 저장소에 업로드
git add schedule/
git commit -m "Add schedule management system"
git push

# 또는 Netlify에 직접 드래그 앤 드롭
```

### 2. 프로그램 실행

브라우저에서 `schedule/index.html` 열기

### 3. 구글 드라이브 연결

1. [구글 드라이브 연결] 버튼 클릭
2. 구글 계정 선택
3. 권한 승인
4. 자동으로 데이터 로드

### 4. 일정 추가

**방법 1: 빈 칸 클릭**
- 원하는 날짜/시간 칸 클릭
- 팝업에서 제목 입력
- [저장] 클릭

**방법 2: 하단 [+추가] 버튼**
- 하단 네비게이션의 [+추가] 클릭
- 모든 정보 입력
- [저장] 클릭

## 📱 사용 가이드

### 일정 추가
1. 캘린더에서 원하는 날짜/시간 클릭
2. 제목, 타입 입력
3. 종일 일정 체크박스 (선택)
4. 시작/종료 날짜, 시간 설정
5. 장소, 메모 입력 (선택)
6. ⭐ 중요 표시 (선택)
7. [저장] 클릭

### 일정 수정
1. 일정 클릭
2. [수정] 버튼 클릭
3. 내용 수정
4. [저장] 클릭

### 일정 삭제
1. 일정 클릭
2. [수정] 버튼 클릭
3. [삭제] 버튼 클릭
4. 확인

### 드래그로 날짜 변경
- 일정을 다른 날짜로 드래그

### 검색
1. 상단 검색창에 키워드 입력
2. [🔍] 버튼 클릭 또는 Enter
3. [✕] 버튼으로 검색 초기화

### 필터
1. [필터] 버튼 클릭
2. 타입/상태 선택
3. [적용] 버튼
4. [초기화]로 전체 보기

### 뷰 변경
1. 하단 [뷰 변경] 버튼
2. 원하는 뷰 선택:
   - 주간 뷰 (권장)
   - 월간 뷰
   - 일간 뷰
   - 리스트 뷰

### 인쇄
- [인쇄] 버튼 클릭
- 프린트 다이얼로그에서 인쇄

## ⚙️ 설정

### 색상 설정
타입별 색상을 원하는 색으로 변경

### 기본 뷰
앱 시작 시 기본 표시될 뷰 선택

### 시간 범위
캘린더에 표시될 시간 범위 설정
- 기본: 09:00 ~ 18:00

### 초기화
모든 설정을 기본값으로 초기화

## 💾 데이터 저장

### 자동 저장
- 일정 추가/수정/삭제 시 3초 후 자동 저장
- 상태 표시: "저장 완료" → "연결됨"

### 수동 저장
- 상단 [💾 저장] 버튼 클릭

### 저장 위치
- 사용자의 구글 드라이브
- 파일명: `schedules.cal`
- 암호화: AES (안전)

---

## 🔒 보안

### 데이터 암호화
- AES 256-bit 암호화
- 암호화 키: 프로그램 내장
- 본인만 복호화 가능

### 개인정보 보호
- 구글 드라이브에만 저장
- 중앙 서버 없음
- 완벽한 프라이버시

## 🐛 문제 해결

### Q: 구글 로그인이 안 돼요
**A:** 
1. 팝업 차단 해제
2. 시크릿 모드에서 시도
3. 다른 브라우저 사용

### Q: 일정이 사라졌어요
**A:**
1. [구글 드라이브 연결] 버튼 클릭
2. 구글 로그인
3. 자동으로 로드됨

### Q: 저장이 안 돼요
**A:**
1. 인터넷 연결 확인
2. 구글 로그인 상태 확인
3. [저장] 버튼 직접 클릭

### Q: 모바일에서 작동이 이상해요
**A:**
- 최신 브라우저 사용 권장
- Chrome, Safari, Samsung Internet
