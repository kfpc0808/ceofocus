## 🚀 **사용 가이드**

### **1. 첫 실행**
1. 브라우저에서 접속
2. [연결] 버튼 클릭
3. 구글 계정 로그인
4. 사용 시작!

### **2. 일정 추가 (빠른 방법)**
1. 원하는 날짜/시간 탭

3. [저장]

1. [📗] 버튼 탭
2. 자동으로 일정 가져옴
3. 회색으로 표시됨

### **4. 뷰 변경**
1. 하단 [뷰] 버튼
2. 원하는 뷰 선택
3. 주간/월간/일간/리스트

### **5. 검색**
1. 상단 검색창
2. 키워드 입력
3. 즉시 필터링

## 🔧 **문제 해결**

### **Q1: 버튼이 너무 작아요**
**A:** 
- 설정 > 접근성 > 큰 텍스트 활성화
- 브라우저 확대 (150% 권장)

### **Q2: 달력이 잘려요 (iOS)**
**A:**
- Safari 설정 > 탭 바 표시 끄기
- 전체 화면 모드 사용
- 홈 화면에 추가 (PWA)

### **Q3: 팝업이 가려져요**
**A:**
- 스크롤해서 아래 버튼 확인
- 가로 모드 사용
- 키보드 닫기 (화면 탭)

### **Q4: 구글 동기화가 안 돼요**
**A:**
1. 인터넷 연결 확인
2. Safari > 설정 > 팝업 허용
3. [🔄] 새로고침 버튼
4. 페이지 새로고침 (F5)
